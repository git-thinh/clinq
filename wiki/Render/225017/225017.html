Supported LINQ Syntax

The following is a list of the operators supported by CLINQ:

Supported operators

  • Where
  • Select
  • SelectMany
  • GroupBy
  • OrderBy (Ascending/Descending)
  • ThenBy (Ascending/Descending)
  • GroupJoin
  • Distinct
  • Except
  • Concat

Aggregation:

  • ContinuousSum
  • ContinuousContains
  • ContinuousFirstOrDefault
  • ContinuousMax
  • ContinuousMin

CLINQ does NOT support:
  • Join
